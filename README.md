



##                                    WanAndroid

WanAndroid，一款每日推荐优质文章App,项目涉及API均来自鸿洋大神的[玩Android](http://www.wanandroid.com/)，基于Material Design + MVP + RxJava2 + Retrofit + Litepal + Glide

出于学习的目的做了这款Material Design风格的简洁的阅读应用，玩*Android*是一个每日推荐20~30篇*Android*优质文章的站点,提供*Android*常用工具和常用网站,并以知识体系的形式展现所有的文章分类。目前包括:  

- [x] 首页模块、banner、常用网站、热门搜索
- [x] 体系模块
- [x] 导航模块
- [x] 项目模块
- [x] TODO模块，增加，删除，修改
- [x] 公众号模块，支持搜索
- [x] 文章详情页
- [x] 收藏，取消收藏
- [x] 登录，注册，登出
- [x] 设置，自动缓存，无图模式，夜间模式，清理缓存
- [x] 关于
